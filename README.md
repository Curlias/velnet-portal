# VELNET Portal Cautivo

Portal cautivo moderno y minimalista para la red Wi-Fi VELNET.

## Características

- Diseño responsive y moderno
- Fondo con degradado tecnológico (azul oscuro a cian)
- Formulario de autenticación seguro
- Listo para desplegar en Vercel o cualquier hosting estático

## Configuración

Antes de desplegar, reemplaza `<IP_DEL_CONTROLADOR>` en el archivo `index.html` con la IP real de tu controlador de red.

## Despliegue

### Vercel
1. Conecta este repositorio a Vercel
2. Despliega automáticamente

### Manual
Simplemente sube el archivo `index.html` a tu servidor web.

## Tecnologías

- HTML5
- CSS3 puro
- Google Fonts (Inter)
